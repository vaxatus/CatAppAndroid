name: devops
description: Set up Docker, CI/CD, project structure, and environment management for multiplatform projects
---

Run the devops-infra droid for infrastructure setup, Docker configuration, CI/CD pipelines, and environment management.

Usage: /devops [setup|review|docker|ci|env] [details]

Examples:
- /devops setup project (scaffold full multiplatform structure)
- /devops docker (create Dockerfiles and docker-compose)
- /devops ci (create GitHub Actions workflows)
- /devops review (audit infrastructure)
