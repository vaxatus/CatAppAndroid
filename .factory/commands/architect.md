---
name: architect
description: Run architecture review, sync API contracts, or manage shared knowledge base
---

Run the project-architect droid to:
1. **Review** — check all platforms against shared contracts for consistency
2. **Sync** — update api-contracts.yaml and data-models.yaml from current code
3. **Add model** — add a new shared model with platform mappings
4. **Add endpoint** — add a new API endpoint to the contract
5. **ADR** — create an Architecture Decision Record

Usage: /architect [review|sync|add-model|add-endpoint|adr] [details]
